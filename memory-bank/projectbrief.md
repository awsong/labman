# LabMan Project Brief

## Overview

LabMan is a laboratory management system designed to track projects, milestones, and progress. It provides a comprehensive dashboard for managing research activities.

## Core Features

- Project management (create, view, edit projects)
- Milestone tracking
- Progress visualization
- User and organization management
- Statistical reporting

## Technical Stack

- Frontend: Vue.js 3 with Vite
- UI: Tailwind CSS
- Backend: Node.js
- Database: SQLite (labman.db)

## Key Components

- Dashboard view
- Project management views
- Milestone tracking
- Progress tracking
- Statistics visualization
- Theme switching
